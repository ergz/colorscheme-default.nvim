# neodef.nvim

## Installation

Lazy:
```lua
{
    'ergz/neodef.nvim',
    lazy = false,
    priority = 1000,
    config = function()
        require('neodef').load()
    end
}
```

Lualine:
```lua
require("lualine").setup({
    options = { theme = "neodef" }
})
```
